README – Proyecto 3: Katas Python

Este proyecto recoge la resolución de 40 katas en Python, cada una diseñada para practicar:

Manipulación de datos

Condicionales y bucles

Funciones y lambdas

map(), filter(), reduce()

Manejo de excepciones

Programación orientada a objetos

Todas las funciones y clases están comentadas para explicar el razonamiento seguido.

+ Pasos seguidos durante el proyecto
1-.  Análisis de los ejercicios

Leí y clasifiqué las 40 katas según su tipo: funciones básicas, lambdas, listas, manejo de errores, clases, etc.

2️-. Desarrollo de cada kata

Fui resolviendo cada ejercicio individualmente, añadiendo:

Comentarios explicativos

Uso correcto de funciones incorporadas (map, filter, reduce)

Validaciones y manejo de errores cuando era necesario

3️-. Implementación de las clases

Desarrollé las clases Arbol y UsuarioBanco siguiendo los métodos obligatorios y añadiendo casos de uso.

4️-. Revisión y orden

Organicé todo el código en un único archivo .py, numerado y comentado para facilitar la lectura y la corrección.

5️-. Preparación del repositorio
